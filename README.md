# 🏝️ 제주 역사 탐험 (Three.js 3D 게임)

제주도 모양의 3D 섬을 캐릭터가 돌아다니며 항일 유적지와 4·3 유적지를 탐방하고 퀴즈를 푸는 게임입니다.

## 실행 방법

ES 모듈을 사용하므로 로컬 서버로 열어야 합니다.

```bash
cd JEJU_BOTH
python3 -m http.server 8123
# 브라우저에서 http://localhost:8123 접속
```

## 조작법

| 키 | 동작 |
|---|---|
| W A S D / 방향키 | 이동 |
| Shift | 달리기 |
| 마우스 드래그 | 시점 회전, 휠로 줌 |
| E | 유적지 살펴보기 (근처에서) |
| 1 ~ 4 | 각 유적지 근처로 빠른 이동 |

## 유적지 (실제 좌표 기반 배치)

| 번호 | 유적지 | 실제 위치 |
|---|---|---|
| 1 | 조천만세동산 | 제주시 조천읍 (33.538, 126.644) |
| 2 | 제주해녀항일운동기념탑 | 제주시 구좌읍 (33.524, 126.861) |
| 3 | 알뜨르 비행장 | 서귀포시 대정읍 (33.204, 126.272) |
| 4 | 제주 4·3 평화공원 | 제주시 봉개동 (33.451, 126.567) |

퀴즈를 모두 맞히면 해당 유적지 마커가 🔴 빨간색 → 🟢 초록색으로 바뀝니다.

## 구글 시트 연동 (Apps Script 방식)

시트를 공개(링크 공유)하지 않아도 되고, 유적지 설명 읽기 + 퀴즈 결과 기록이 모두 됩니다.
퀴즈 문제는 게임에 내장된 것([js/data.js](js/data.js))을 사용합니다.

### 1단계 — 시트에 `sites` 탭 작성

새 구글 시트를 만들고 탭 이름을 `sites`로 지정한 뒤 아래처럼 작성합니다 (1행은 헤더).

| site_id | name | history |
|---|---|---|
| jocheon | 조천만세동산 | 1919년 3월 21일, 제주에서 처음으로... |
| haenyeo | 제주해녀항일운동기념탑 | 1931~1932년 구좌읍 일대에서... |
| alddreu | 알뜨르 비행장 | '알뜨르'는 제주말로... |
| peace43 | 제주 4·3 평화공원 | 제주 4·3 사건으로 희생된... |

- `site_id`는 반드시 `jocheon` / `haenyeo` / `alddreu` / `peace43` 중 하나여야 합니다.
- `name`이나 `history`를 비워두면 그 항목은 내장 내용이 유지됩니다.

### 2단계 — Apps Script 붙여넣기

시트에서 **확장 프로그램 → Apps Script** 를 열고, 기본 코드를 지운 뒤
프로젝트의 [apps-script.gs](apps-script.gs) 파일 내용 **전체**를 붙여넣고 저장합니다.

### 3단계 — 웹 앱으로 배포

1. Apps Script 편집기 우측 상단 **배포 → 새 배포**
2. 유형: **웹 앱**
3. **실행 계정: 나** / **액세스 권한: 모든 사용자** ← 중요!
4. 배포를 누르고 권한을 승인하면 나오는 **웹 앱 URL** 복사
   (`https://script.google.com/macros/s/…/exec` 형태)

### 4단계 — 게임에 연결

[js/config.js](js/config.js)에 URL을 붙여넣습니다.

```js
export const CONFIG = {
  APPS_SCRIPT_URL: "https://script.google.com/macros/s/…/exec",  // ← 여기에
  SHEET_ID: "",
};
```

게임을 새로고침해서 시작 화면 하단에 `✅ 구글 시트 연결됨 · Apps Script`가 뜨면 성공입니다.
이후 플레이어가 퀴즈를 완료할 때마다 시트의 `log` 탭에 시각·플레이어·유적지·오답 횟수가 자동으로 쌓입니다.

> 참고: 시트 내용을 고친 뒤에는 게임을 새로고침만 하면 반영됩니다.
> 단, apps-script.gs 코드를 수정했다면 **배포 → 배포 관리 → 수정(연필) → 새 버전**으로 다시 배포해야 합니다.
